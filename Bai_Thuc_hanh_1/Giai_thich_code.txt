# ESP32 Đọc Cảm Biến Ánh Sáng LDR Qua Chân D0

## Mô tả

Dự án sử dụng ESP32 để đọc tín hiệu từ cảm biến ánh sáng LDR thông qua chân D0 (Digital Output). ESP32 sẽ xác định môi trường có ánh sáng hay không và hiển thị kết quả trên Serial Monitor.

## Phần cứng sử dụng

* ESP32 DevKit V1
* Module cảm biến LDR (có chân D0)
* Dây nối

## Kết nối phần cứng

| LDR Module | ESP32 |
| ---------- | ----- |
| VCC        | 3.3V  |
| GND        | GND   |
| D0         | GPIO2 |

## Chương trình

```cpp
#define LDR_PIN 2

void setup() {
  Serial.begin(115200);
  pinMode(LDR_PIN, INPUT);
}

void loop() {
  int ldrState = digitalRead(LDR_PIN);

  if (ldrState == HIGH) {
    Serial.println("Khong co anh sang");
  } else {
    Serial.println("Co anh sang");
  }

  delay(500);
}
```

## Nguyên lý hoạt động

1. ESP32 khởi tạo giao tiếp Serial với tốc độ 115200 baud.
2. GPIO2 được cấu hình ở chế độ INPUT để nhận tín hiệu từ chân D0 của cảm biến.
3. Trong vòng lặp `loop()`, ESP32 đọc trạng thái logic của GPIO2 bằng hàm `digitalRead()`.
4. Nếu giá trị đọc được là HIGH, chương trình in ra thông báo "Khong co anh sang".
5. Nếu giá trị đọc được là LOW, chương trình in ra thông báo "Co anh sang".
6. Sau mỗi lần đọc, chương trình chờ 500 ms rồi tiếp tục lặp.

## Giá trị đọc được

| Giá trị  | Ý nghĩa                                             |
| -------- | --------------------------------------------------- |
| HIGH (1) | Không có ánh sáng hoặc ánh sáng dưới ngưỡng cài đặt |
| LOW (0)  | Có ánh sáng vượt ngưỡng cài đặt                     |

> Lưu ý: Một số module LDR có thể đảo ngược mức HIGH và LOW tùy theo nhà sản xuất. Nếu kết quả hiển thị không đúng thực tế, hãy đổi nội dung hai nhánh `if` và `else`.

## Kết quả mong đợi

Khi môi trường có ánh sáng:

```text
Co anh sang
Co anh sang
Co anh sang
```

Khi che cảm biến hoặc môi trường tối:

```text
Khong co anh sang
Khong co anh sang
Khong co anh sang
```

## Ứng dụng

* Hệ thống đèn tự động.
* Giám sát cường độ ánh sáng.
* Nhà thông minh (Smart Home).
* Hệ thống cảnh báo ngày/đêm.
